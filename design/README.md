## Design Direction

**Style:** Nocturne + Minimal Tech (80/20)
**Glow Color:** Plasma Cyan (`#06B6D4`)
**Typefaces:** Inter (UI) + JetBrains Mono (code/conversations)
**Canvas:** `#0F0F14` (near-black, not pure black)

This is a dark-native dev tool. No light mode. Terminal kinship. Information density over decoration. Every bright pixel earns its place.
